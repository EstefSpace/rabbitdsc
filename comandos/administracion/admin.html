    Bienvenidas


    Este es el primer comando de esta categoria, rabbit bot te incluye un mensaje de bienvenida a los nuevos usuarios de tu servidor con imagen personalizada, cualquier error puedes reportarlo en el servidor de soporte.


    /bienvenidas setup

    Activa el sistema de bienvenidas en tu servidor, este subcomando cuenta con 2 opciones.




    Opciones Requerido
    canal Sí
    imagen No

    /bienvenidas disable


    Desactiva el sistema de bienvenidas en tu servidor, este subcomando no necesita ninguna opcion.

    Verificacion


    Este es el tercer comando de esta categoria, rabbit bot verificara que el usuario sea real y no un robot, cualquier error puedes reportarlo en el servidor

    Activa el sistema de verifiacion en tu servidor, este subcomando cuenta con 4 opciones.




    Opciones Requerido
    canal Sí

    titulo No
    descripcion No

    /verificacion disable


    Desactiva el sistema de verificacion en tu servidor, este subcomando no necesita ninguna opcion.

    Tickets


    Este es el cuarto comando de esta categoria, rabbit bot hara un canal donde tienes varias opciones del ticket, cualquier error puedes reportarlo en el servidor


    /tickets setup

    Activa el sistema de tickets en tu servidor, este subcomando cuenta con 6 opciones.




    Opciones Requerido
    canal Sí
    rol Sí
    titulo Sí
    descripcion Sí
    color Sí
    miniatura Sí

    /tickets disable


    Desactiva el sistema de tickets en tu servidor, este subcomando no necesita ninguna opcion.
